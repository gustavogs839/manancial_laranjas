# Manancial Laranjas — React (scaffold)

Como rodar:

1. Abra terminal em `react-app`.
2. Instale dependências:

```bash
npm install
```

3. Inicie em modo dev:

```bash
npm run dev
```

O app ficará disponível em http://localhost:5173 por padrão.

Observações:
- Este scaffold é um ponto de partida. A partir daqui podemos migrar toda a lógica antiga para componentes adicionais (export Elgin, impressão térmica via backend, etc).
